# Mars Weather Prediction

Predicting maximum surface temperature on Mars using machine learning. Based on Curiosity Rover REMS data (August 2012 - February 2018).

## Models Implemented

- Linear Regression (baseline)
- MLP Neural Network (hidden layers: 100, 50)
- Random Forest Regressor (100 estimators)
- Random Forest + PCA (3 components)

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R-squared (R²)
